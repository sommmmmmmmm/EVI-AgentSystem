## 🚨 Critical Disclaimers

### ⚠️ 투자 권고가 아님
```
본 보고서는 정보 제공 목적이며, 
특정 종목의 매수/매도를 권고하는 것이 아닙니다.

투자 결정은 반드시 본인의 판단과 책임하에 이루어져야 합니다.
```

### ⚠️ 재무 자문이 아님
```
본 시스템은 금융투자업 라이선스가 없으며,
법적으로 투자 자문을 제공할 수 없습니다.

전문 금융 자문이 필요한 경우 
공인된 투자자문사를 이용하시기 바랍니다.
```

## 📋 1. 데이터 한계

### 1.1 데이터 지연
| 데이터 소스 | 업데이트 주기 | 지연 시간 |
|------------|-------------|----------|
| **뉴스 (GNews)** | 실시간 | ~1시간 |
| **DART 공시** | 당일 | ~1일 |
| **SEC 공시** | 당일 | ~1일 |
| **재무제표** | 분기 | ~45-90일 |
| **Yahoo Finance** | 일일 | ~1일 |

**한계**:
- ❌ 실시간 주가 변동 반영 안 됨
- ❌ 장 마감 후 공시는 다음날 반영
- ❌ 분기 실적은 발표 후에야 반영

**예시**:
```
시나리오: Tesla가 오늘 실적 발표 (주가 +15%)
본 시스템: 내일 뉴스 수집 후 반영
→ 투자 타이밍을 놓칠 수 있음
```

### 1.2 데이터 커버리지 부족
| 구분 | 커버리지 | 부족한 부분 |
|------|---------|-----------|
| **한국 상장사** | 50+ | 코스닥 소형주 ❌ |
| **미국 상장사** | 30+ | 나스닥 소형주 ❌ |
| **중국/유럽** | 10+ | 대형주만 ❌ |
| **비상장사** | 제한적 (5개) | 대부분 누락 ❌ |

**한계**:
- ❌ 소형주/중소형주 분석 부족
- ❌ 신생 기업 (IPO 후 1년 미만) 데이터 부족
- ❌ 비상장 유망 기업 분석 불가

**예시**:
```
실제 고수익 종목: 코스닥 소형 배터리 소재 기업 (+200%)
본 시스템: 데이터 부족으로 발견 불가
→ Hidden Gem 발굴 어려움
```

### 1.3 데이터 품질 문제
**문제점**:
1. **웹 검색 결과의 신뢰도**
   - Tavily 검색 점수: 0.4~0.8 (완벽하지 않음)
   - 잘못된 정보 포함 가능성
   - 출처 검증 한계

2. **재무 데이터 일관성**
   - DART vs SEC vs Yahoo Finance 차이
   - 회계 기준 차이 (K-IFRS vs US-GAAP)
   - 환율 영향 미고려

3. **뉴스 편향**
   - 대형주 뉴스에 편중
   - 긍정적 뉴스 과다 (홍보성)
   - 부정적 뉴스 누락 가능

**예시**:
```
뉴스: "LG에너지솔루션, 대규모 투자 발표" (긍정적)
누락: 소액주주 소송, 내부고발 (부정적)
→ 편향된 분석 위험
```

## 📋 2. 분석 깊이 한계

### 2.1 정량 분석 한계
**현재 분석 지표**:
```python
{
  "재무비율": ["ROE", "부채비율", "영업이익률", "유동비율"],
  "성장성": ["매출증가율", "영업이익증가율"],
  "수익성": ["ROE", "ROA", "ROIC"],
  "안정성": ["부채비율", "유동비율", "이자보상배율"]
}
```

**부족한 분석**:
- ❌ **Valuation**: PER, PBR, EV/EBITDA 분석 미흡
- ❌ **Cash Flow**: FCF, OCF 분석 없음
- ❌ **Margin Trend**: 마진 추세 분석 없음
- ❌ **Peer Comparison**: 동종 업계 상대 비교 부족
- ❌ **Technical Analysis**: 차트, RSI, MACD 등 기술적 분석 전무

**예시**:
```
회사 A: ROE 15% (좋음)
실제: PER 50배 (고평가) → 매수 부적절
본 시스템: PER 미분석 → 과대평가 위험
```

### 2.2 정성 분석 한계
**현재 분석 방식**:
- LLM 기반 뉴스/공시 텍스트 분석
- 키워드 기반 트렌드 추출
- 웹 검색 기반 리스크 평가

**한계**:
1. **경영진 분석 부족**
   - CEO 역량, 지배구조
   - 이사회 독립성
   - 소액주주 친화도

2. **산업 분석 깊이 부족**
   - Porter's 5 Forces 미적용
   - SWOT 분석 피상적
   - 경쟁 우위 (Moat) 분석 부족

3. **거시 경제 고려 부족**
   - 금리, 환율, 유가 영향 미분석
   - 지정학적 리스크 미고려
   - 정책 변화 (보조금, 세제) 반영 미흡

**예시**:
```
분석: "Tesla는 EV 시장 리더"
누락: 
- 중국 보조금 삭감 리스크
- 유럽 탄소배출 규제 완화
- 미국 IRA 법안 변경
→ 거시 변수 리스크 간과
```

### 2.3 AI/LLM 한계
**LLM 분석의 한계**:
1. **Hallucination (환각)**
   ```
   LLM 출력: "회사 A는 시장 점유율 30%"
   실제: 출처 없는 추측, 잘못된 정보
   ```

2. **Training Data Cutoff**
   ```
   GPT-4o-mini: 2023년 10월까지 학습
   → 최신 산업 트렌드 미반영
   ```

3. **수치 계산 약점**
   ```
   LLM: "ROE가 높아서 투자 매력적"
   실제: ROE 15% (업계 평균 20%) → 낮음
   → 상대적 비교 부족
   ```

4. **Fallback 품질**
   ```
   Plan A (LLM): 고품질 분석 ⭐⭐⭐
   Plan B (재무 데이터): 중간 품질 ⭐⭐
   Plan C (템플릿): 낮은 품질 ⭐
   → API 실패 시 품질 저하
   ```

## 📋 3. 투자 판단 한계

### 3.1 목표가 산정 한계
**현재 방식**:
```python
# Plan A: LLM 추정
target_price = llm.generate("목표가를 추정해주세요")

# Plan B: 단순 상승률 (20%)
target_price = current_price * 1.2

# Plan C: None
target_price = None
```

**한계**:
- ❌ DCF (Discounted Cash Flow) 모델 미사용
- ❌ 애널리스트 컨센서스 미반영
- ❌ Multiple 분석 (PER, PBR) 미사용
- ❌ 상승 여력 산정 근거 부족

**예시**:
```
본 시스템: "목표가 50,000원 (현재가 대비 +20%)"
전문가: DCF 기반 내재가치 35,000원 (-30%)
→ 과대평가 위험
```

### 3.2 투자 시점 판단 불가
**현재 제공**:
- 투자 기간 (단기/중기/장기)
- 투자 근거

**제공 안 함**:
- ❌ 매수 타이밍 (지금? 하락 시?)
- ❌ 분할 매수 전략
- ❌ 손절/익절 기준 가격
- ❌ 리밸런싱 시기

**예시**:
```
보고서: "Tesla 장기 투자 추천"
현실: 
- 현재 고점 (PER 80배)
- 3개월 후 20% 하락
→ 매수 타이밍 실패
```

### 3.3 포트폴리오 최적화 한계
**현재 방식**:
```python
# 투자 점수 기반 가중치 배분
weights = normalize([score1, score2, ...])
# 예: [30%, 25%, 20%, 15%, 10%]
```

**한계**:
- ❌ 리스크-수익률 최적화 (Markowitz) 미사용
- ❌ 상관관계 고려 안 됨 (분산 효과 미흡)
- ❌ 개인 투자자 프로필 미반영
  - 연령, 자산 규모
  - 위험 감수 성향
  - 투자 목적 (은퇴, 재테크)

**예시**:
```
포트폴리오: [LG에너지 30%, SK On 25%, 삼성SDI 20%]
문제: 3개 모두 한국 배터리 → 상관관계 0.9
→ 분산투자 효과 없음
```

## 📋 5. 법적/규제 한계

### 5.1 금융투자업법 제한
**한국 금융투자업법**:
```
제6조 (금융투자업의 인가)
투자자문업, 투자일임업을 영위하려면 금융위원회의 인가 필요
```

**본 시스템**:
- ❌ 금융투자업 인가 없음
- ❌ 투자자문 제공 불가
- ⚠️ 정보 제공만 가능 (자문 아님)

**결과**:
```
할 수 있는 것:
✅ "LG에너지솔루션의 ROE는 15%입니다"
✅ "배터리 시장은 성장하고 있습니다"

할 수 없는 것:
❌ "LG에너지솔루션을 매수하세요"
❌ "포트폴리오의 30%를 투자하세요"
❌ "목표가 50만원까지 보유하세요"
```

### 5.2 면책 조항 필수
**필수 명시사항**:
```
⚠️ 본 보고서는 투자 권유가 아닙니다
⚠️ 투자 손실의 책임은 투자자 본인에게 있습니다
⚠️ 과거 데이터는 미래 수익을 보장하지 않습니다
⚠️ 정보의 정확성을 보장하지 않습니다
```

### 5.3 증권신고서/투자설명서 아님
**본 보고서는**:
- ❌ 금융감독원 제출 서류 아님
- ❌ 법적 효력 없음
- ❌ 감사인의 검토 거치지 않음

## 📋 6. 기술적 한계

### 6.1 API 의존성
**Single Point of Failure**:
```python
의존 API:
- OpenAI API (LLM) → 중단 시 품질 저하
- Tavily API (웹 검색) → 중단 시 정보 부족
- DART API (공시) → 중단 시 한국 기업 분석 불가
- SEC API (공시) → Rate Limit 시 미국 기업 누락
```

**한계**:
- ⚠️ 외부 API 장애 시 보고서 품질 저하
- ⚠️ Rate Limit 도달 시 데이터 수집 중단
- ⚠️ API 비용 부담 (대규모 분석 시)

### 6.2 LangGraph 버그
**현재 상태**:
```python
# LangGraph 사용 불가
# KeyError: '__start__'

# 대안: 수동 실행
for agent in agents:
    result = agent.process(state)
    state.update(result)
```

**한계**:
- ❌ 워크플로우 자동화 제한
- ❌ 병렬 처리 불가
- ❌ 체크포인팅 불가 (중단 후 재시작 어려움)

### 6.3 확장성 한계
**현재 제약**:
```
- 동시 분석 기업 수: ~30개 (메모리 제약)
- 실행 시간: ~14분 (순차 실행)
- 보고서 길이: ~5000 토큰 (LLM 제약)
```

**대규모 분석 불가**:
- ❌ 코스피 전체 (800+) 동시 분석 불가
- ❌ 글로벌 EV 업체 전체 (200+) 분석 불가
- ❌ 실시간 모니터링 불가
