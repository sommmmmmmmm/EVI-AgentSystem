## 1. 데이터 한계

### 데이터 지연
- **뉴스 데이터**: 1-2시간 지연
- **공시 데이터**: 1영업일 지연
- **재무제표**: 45-90일 지연

### 커버리지 제한
- **대형주 중심**: 소형주 분석 부족
- **비상장 기업**: 최대 5개로 제한

### 데이터 품질
- **웹 검색 신뢰도**: 0.4~0.8 (불완전)
- **재무 데이터 불일치**: DART, SEC, Yahoo Finance 간 차이
- **뉴스 편향**: 대형주 과다 대표, 긍정적 뉴스 편중

```

GM: 신뢰도 0.80
```

**문제점:**

**실제 분류:**
